==========================================
FBPlayer - Feedback Animation System
==========================================

🔧 Requirements (依赖项)
--------------------------
This package requires the following third-party plugins:

1. Odin Inspector (by Sirenix)
   - Used for enhanced editor UI and attributes.
   - Must be imported before using this package.
   - Available at: https://assetstore.unity.com/packages/tools/utilities/odin-inspector-89041

2. DOTween (by Demigiant)
   - Used for animation and tweening.
   - Available for free at: https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676

Please make sure both plugins are imported and installed in your project **before using this asset**.

⚠️ 如果未导入 Odin Inspector 或 DOTween，本插件将无法正常运行。

📘 Instructions (使用说明)
--------------------------
- Import Odin Inspector and DOTween into your Unity project first.
- Then import this package.
- You will find the FBPlayer component ready to use on GameObjects.

📌 注意事项
--------------------------
- 本资源依赖 Odin Inspector 和 DOTween 插件。
- 使用前请先确保这两个插件已导入到项目中。
- 如未安装，将导致脚本无法使用或 Inspector 异常。

Thank you for using FBPlayer! 🙌
